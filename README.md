## 📖 Опис проєкту
Це портфоліо було розроблене мною для презентації моїх навчальних досягнень, виконаних лабораторних робіт та проєктів. Під час роботи я приділяла особливу увагу чистоті коду, адаптивності верстки та сучасному візуальному стилю.

## 🛠 Технології та навички
У процесі створення цього портфоліо я використала та закріпила наступні знання:
* **HTML5 & CSS3:** Побудова семантичної структури та стилізація елементів.
* **Flexbox/Grid:** Створення адаптивної сітки, яка коректно відображається на різних пристроях.

## 🚀 Як я це зробила 
1. **Ініціалізація:** Я створила структуру папок та базовий файл `index.html`.
2. **Верстка:** Використала семантичні теги HTML5 для правильної розмітки контенту.
3. **Стилізація:** Застосувала CSS3 для надання сайту фірмового стилю.
4. **Навчання:** У процесі виконання я навчилася працювати з Git та налаштовувати GitHub Pages для публікації сайту.
5. **Деплой:** Налаштувала автоматичне відображення сторінки через GitHub Pages.
